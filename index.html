Durante meu treinamento na Smart Consulting, tive a oportunidade de desenvolver uma aplicação de vendas utilizando as tecnologias da Power Platform, incluindo Power Apps, Power Automate e Power BI. Trabalhei em estreita colaboração com desenvolvedores experientes para criar interfaces intuitivas e apliquei plug-ins personalizados em C# para validações específicas. Automatizei processos de vendas, como a aprovação de pedidos via e-mail e Microsoft Teams, tornando o fluxo de trabalho mais ágil e eficiente. Além disso, elaborei dashboards detalhados com Power BI, permitindo uma visualização clara e instantânea dos dados, o que facilitou a tomada de decisões estratégicas. Esse projeto também me proporcionou um ótimo entendimento do Dynamics 365 CRM. Ao integrar Dynamics 365 com a Power Platform, consegui não apenas otimizar o desempenho das vendas, mas também obter insights valiosos sobre os clientes. O Dynamics 365 CRM é uma poderosa plataforma para gestão de relacionamento com o cliente, e quando combinada com as ferramentas da Power Platform, permite criar soluções altamente personalizadas e escaláveis. A integração entre estas tecnologias oferece um ecossistema unificado onde dados, processos e pessoas estão perfeitamente conectados, resultando em uma performance superior e uma compreensão mais profunda das necessidades dos clientes.

Tecnologias usadas

Desenvolvi um projeto de back-end focado na implementação de um sistema robusto de login de usuário. Este sistema inclui validações detalhadas para o registro de contas, garantindo a criação de senhas fortes e a integridade dos dados. Configurei um banco de dados seguro para armazenar as contas registradas, com as senhas criptografadas utilizando Bcrypt. Todo o fluxo de autenticação foi projetado para assegurar que o login seja realizado de forma segura, seguindo as melhores práticas de segurança da informação. Utilizei uma stack tecnológica completa, incluindo Next.js para estruturar o projeto com rotas dinâmicas, pages e API routes, aproveitando seu sistema de renderização híbrido para melhorar a performance. O desenvolvimento foi realizado com React, Node.js e JavaScript, e o código foi gerenciado com Git e hospedado no GitHub. O ambiente de desenvolvimento foi o Visual Studio Code, e a aplicação está hospedada na Vercel, garantindo um deployment contínuo e fácil.
